CentOS与Redis

2022年1月12日
| 服务器
| 本文阅读量 次
文章目录

本文介绍如何在 CentOS 上安装、配置、启动、关闭和卸载 Redis。

安装

下载Redis安装包

本文介绍两种方法：

  1. 在 Redis 官网下载最新版本的 redis-6.2.6。

  2. 通过命令下载：

1$ wget http://download.redis.io/releases/redis-6.2.6.tar.gz

安装Redis编译的C环境

由于 Redis 是由 C 语言写成的，所以需要 gcc 编译工具。

1$ yum install gcc-c++

查看 gcc 是否安装，可以使用：

1$ rpm -q gcc

如果出现类似下面的输出，则说明 gcc 已经正确安装在 CentOS 服务器上了。

1$ rpm -q gcc
2gcc-8.5.0-7.el8.x86_64

解压安装包并进入文件夹

  1. 解压安装包
1$ tar -zxvf redis-6.2.6.tar.gz
  1. 进入 redis-6.2.6 目录
1$ cd redis-6.2.6

编译Redis

安装包中都是 C 语言编写的源程序，需要使用 make 命令进行编译。

1$ make
2...
3Hint: It's a good idea to run 'make test' ;)
4make[1]: Leaving directory '/home/imw7/pkg/redis-6.2.6/src'

出现类似输出结果则表示 make 成功。

安装编译后的文件到指定目录

编译好之后，需要将其安装到指定的目录，方便使用。

1$ make PREFIX=/usr/local/redis install

注意：PREFIX 必须大写，同时会自动为我们生成 redis 目录，并将结果安装到此目录。

安装完成之后，在 /usr/local/redis/bin 目录下会生成几个文件：

1bin
2├── redis-benchmark
3├── redis-check-aof -> redis-server
4├── redis-check-rdb -> redis-server
5├── redis-cli
6├── redis-sentinel -> redis-server
7└── redis-server

配置

将配置文件移动到指定位置

  1. 创建目录 etc
1$ mkdir /usr/local/redis/etc
  1. 复制配置文件到 etc 目录
1$ cd redis-6.2.6
2$ cp redis.conf /usr/local/redis/etc
  1. 进入 /usr/local/redis 修改 redis.conf 配置文件
1$ vim redis.conf

Redis配置默认必须修改

在 vim 中，使用 /内容 来搜索要找的内容，如 /daemonize，可快速定位要找的位置。

  • 配置 Redis 为后台启动
1daemonize no # 修改为 daemonize yes
  • 开启外网访问
1bind 127.0.0.1 # 将这行注释掉，在最前面加“#”，即可注释
  • 配置密码
1requirepass password

Redis启动

在执行命令之前，首先关闭防火墙，否则可能导致无法连接数据库。

1$ systemctl stop firewalld

服务端启动

1$ /usr/local/redis/bin/redis-server /usr/local/redis/etc/redis.conf

查看进程

1$ ps -ef | grep redis
2root        1267       1  0 11:58 ?        00:00:03 /usr/local/redis/bin/redis-server *:6379
3root        1857    1763  0 12:46 pts/0    00:00:00 grep --color=auto redis

客户端启动

本地启动

1$ cd /usr/local/redis/bin
2$ ./redis-cli # 没密码
3$ ./redis-cli -a password # 有密码
4$ exit # 退出

远程调用服务

1$ redis-cli -h host -p port -a password  # redis-cli -h IP地址 -p 端口 -a 密码

go代码设置必须设置密码才能连接

 1import (
 2	"fmt"
 3	"github.com/gomodule/redigo/redis"
 4	"log"
 5)
 6
 7func main() {
 8	// 1.连接数据库
 9	conn, err := redis.Dial("tcp", "127.0.0.1:6379")
10	if err != nil {
11		log.Fatal("redis dial failed, err:", err)
12	}
13	defer func(conn redis.Conn) {
14		err := conn.Close()
15		if err != nil {
16			return
17		}
18	}(conn)
19
20	// 2.设置密码
21	if _, err := conn.Do("AUTH", "password"); err != nil {
22		log.Fatal(err)
23	}
24
25	// 3.操作数据库
26	reply, err := conn.Do("set", "foo", "bar")
27
28	// 4.回复助手类函数，确定成具体的数据类型
29	str, err := redis.String(reply, err)
30
31	fmt.Println(str, err)
32}

Redis关闭

第一种关闭方式（断电、非正常关闭，容易数据丢弃）：

1$ ps -ef | grep -i redis # 查询PID
2$ kill -9 PID

第二种关闭方式（正常关闭，数据保存）：

1$ ./bin/redis-cli shutdown

卸载

  1. 先把 Redis 服务关闭
  2. 再把 /usr/local/redis/bin/ 目录下的与 Redis 相关的文件删除即可。
