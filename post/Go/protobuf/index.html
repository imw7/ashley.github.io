protobuf是一种高效的数据格式，平台无关、语言无关、可扩展，可用于 RPC 系统和持续数据存储系统。

protobuf

protobuf介绍

Protobuf是Protocol Buffer的简称，它是Google公司于2008年开源的一种高效的平台无关、语言无关、可扩展的数据格式，目前Protobuf作为接口规范的描述语言，可以作为Go语言RPC接口的基础工具。

protobuf使用

protobuf是一个与语言无关的一个数据协议，所以我们需要先编写IDL文件然后借助专用工具生成指定语言的代码，从而实现数据的序列化与反序列化过程。

大致开发流程如下： 1. IDL编写 2. 生成指定语言的代码 3. 序列化和反序列化

编译器安装

protoc

protobuf协议编译器是用c++编写的，根据自己的操作系统下载对应版本的protoc编译器：https://github.com/protocolbuffers/protobuf/releases，解压后拷贝到GOPATH/bin目录下。

protoc-gen-go

安装生成Go语言代码的工具

1go get -u github.com/golang/protobuf/protoc-gen-go

编写IDL代码

在protobuf_demo/address目录下新建一个名为person.proto的文件具体内容如下：

 1// 指定使用protobuf版本
 2// 此处使用v3版本
 3syntax = "proto3";
 4
 5// 包名，通过protoc生成go文件
 6package address;
 7
 8// 性别类型
 9// 枚举类型第一个字段必须为0
10enum GenderType {
11    SECRET = 0;
12    FEMALE = 1;
13    MALE = 2;
14}
15
16// 人
17message Person {
18    int64 id = 1;
19    string name = 2;
20    GenderType gender = 3;
21    string number = 4;
22}
23
24// 联系簿
25message ContactBook {
26    repeated Person persons = 1;
27}

生成go语言代码

在protobuf_demo/address目录下执行以下命令。

1address $ protoc --go_out=. ./person.proto

此时在当前目录下会生成一个person.pb.go文件，我们的Go语言代码里就是使用这个文件。 在protobuf_demo/main.go文件中：

 1package main
 2
 3import (
 4	"fmt"
 5	"io/ioutil"
 6	"github.com/golang/protobuf/proto"
 7	"github.com/imw7/studygo/code_demo/protobuf_demo/address"
 8)
 9
10// protobuf demo
11
12func main() {
13	var cb address.ContactBook
14
15	p1 := address.Person{
16		Name:   "张三",
17		Gender: address.GenderType_MALE,
18		Number: "7878778",
19	}
20	fmt.Println(p1)
21	cb.Persons = append(cb.Persons, &p1)
22	// 序列化
23	data, err := proto.Marshal(&p1)
24	if err != nil {
25		fmt.Printf("marshal failed,err:%v\n", err)
26		return
27	}
28	ioutil.WriteFile("./proto.dat", data, 0644)
29
30	data2, err := ioutil.ReadFile("./proto.dat")
31	if err != nil {
32		fmt.Printf("read file failed, err:%v\n", err)
33		return
34	}
35	var p2 address.Person
36	proto.Unmarshal(data2, &p2)
37	fmt.Println(p2)
38}
